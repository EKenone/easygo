### 目录结构说明
> 以下是主目录结构，如果需求自行扩展

    api --------------------------------------------- 服务协议文件的目录
    config ------------------------------------------ 配置文件存放目录
    cmd --------------------------------------------- 可执行文件目录
    internal ---------------------------------------- 内部私有代码
        app ----------------------------------------- 应用实现目录
             api ------------------------------------ 对外接口
             model ---------------------------------- 存放数据库模型
             service -------------------------------- 业务实现的地方
        config -------------------------------------- 配置文件解析的结构体的目录
        init ---------------------------------------- 唯一可以使用init方法的目录
        plugins ------------------------------------- 业务使用的插件包目录
        utils --------------------------------------- 辅助函数存放目录
    log --------------------------------------------- 日志文件目录
    pkg --------------------------------------------- 公用组件包存放的地方
    test -------------------------------------------- 单元测试
